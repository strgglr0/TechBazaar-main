## Introduction

**TechBazaar** is a comprehensive e-commerce platform specializing in technology products including smartphones, laptops, monitors, headphones, and accessories. This platform provides a seamless shopping experience with advanced search capabilities, product filtering, shopping cart management, and robust administrative tools for inventory and order management.

This user manual is divided into two main sections:
- **Customer User Manual**: For end-users who browse and purchase products
- **Admin User Manual**: For administrators who manage products, orders, and analytics

---

## System Requirements

To use TechBazaar, ensure your system meets the following requirements:
- **Web Browser**: Modern browser (Chrome 90+, Firefox 88+, Safari 14+, Edge 90+)
- **Internet Connection**: Stable broadband connection recommended
- **JavaScript**: Must be enabled in browser settings
- **Screen Resolution**: Minimum 1024x768 (responsive design supports mobile devices)
- **Cookies**: Must be enabled for authentication

# a. Customer User Manual

This section explains how customers can use the TechBazaar system to browse, search, and purchase products.

## 1. Getting Started as a Customer

### 1.1 Accessing TechBazaar
1. Open your web browser
2. Navigate to the TechBazaar URL
3. The home page displays featured products and category options
4. You can browse as a guest or create an account for full features

### 1.2 Customer Account Types

**Guest Users (No Account Required)**
- Browse all products
- View product details
- Search and filter products
- Cannot add to cart or make purchases

**Registered Users (Account Required)**
- All guest features
- Shopping cart functionality
- Place orders and checkout
- View order history
- Track past purchases

### 1.3 Creating a Customer Account
1. Click **Sign Up** button in the top navigation bar
2. Complete the registration form:
   - **Full Name**: Enter your complete name
   - **Email Address**: Enter a valid email (used for login)
   - **Password**: Create a password (minimum 6 characters)
3. Click **Create Account** button
4. System automatically logs you in
5. You can now access all customer features

### 1.4 Logging Into Your Account
1. Click **Login** button in the navigation bar
2. Enter your registered email address
3. Enter your password
4. Click **Sign In** button
5. You will be redirected to the home page


### 1.5 Logging Out
1. Click your profile icon or name in the top navigation bar
2. Select **Logout** from the dropdown menu
3. You will be logged out and redirected to the home page

## 2. Browsing and Searching Products

### 2.1 Home Page Overview
The home page provides:
- **Hero Banner**: Featured categories with quick access buttons
- **Product Grid**: Displays all available products with images and prices
- **Navigation Bar**: Access to search, cart, and user account
- **Category Filters**: Quick filter buttons for product categories
- **Sort Options**: Dropdown to organize products

### 2.2 Viewing Product Details
1. Click on any product card from the home page
2. Product detail page displays:
   - **Product Images**: High-quality product photos
   - **Product Name and Brand**: Clear identification
   - **Price**: Current selling price in Philippine Peso (‚Ç±)
   - **Description**: Detailed product information
   - **Specifications**: Technical details and features
   - **Stock Status**: Availability information
   - **Reviews and Ratings**: Customer feedback (if available)
   - **Add to Cart Button**: Quick purchase option
3. Scroll down to view related or similar products
4. Click **Back** or use browser back button to return to browsing

### 2.3 Using the Search Function
1. Locate the search bar at the top of any page
2. Type your search query:
   - Product names (e.g., "iPhone", "MacBook")
   - Brand names (e.g., "Samsung", "Dell")
   - Product types (e.g., "laptop", "headphones")
   - Keywords from descriptions
3. Press **Enter** or click the search icon (magnifying glass)
4. View search results page showing matching products
5. Search results display:
   - Number of products found
   - Matching products with highlighted relevance
   - Same filtering and sorting options as home page

**Search Tips:**
- Use specific product names for exact matches
- Use general terms for broader results
- Check spelling for best results
- Try different keywords if no results found

### 2.4 Filtering Products

#### Category Filter
Select from these categories:
- **Smartphones**: Mobile phones and accessories
- **Laptops**: Portable computers and notebooks
- **Monitors**: Computer displays and screens
- **Headphones**: Audio devices and earphones
- **Accessories**: Cables, chargers, cases, and more

**How to Use:**
1. Click on category buttons in the hero section, OR
2. Use the category filter in the left sidebar
3. Products update automatically to show selected category

#### Price Range Filter
1. Locate the price slider in the left sidebar
2. Drag the left handle to set minimum price
3. Drag the right handle to set maximum price
4. Price range displays as: ‚Ç±[min] - ‚Ç±[max]
5. Products filter automatically as you adjust
6. Available range: ‚Ç±0 to ‚Ç±200,000+

#### Brand Filter
1. Find the brand selection in the left sidebar
2. View list of available brands (Apple, Samsung, Dell, HP, Sony, etc.)
3. Click on brand names to filter by specific manufacturer
4. Select multiple brands to view products from different manufacturers
5. Clear selection to view all brands

#### Rating Filter
1. Use the rating filter in the sidebar
2. Select minimum star rating (1 to 5 stars)
3. Only products meeting or exceeding the rating threshold will display
4. Higher ratings show premium quality products

### 2.5 Sorting Products
Use the sort dropdown (top right of product grid) to organize products:

- **Featured** (Default): Curated selection and popular items
- **Name (A-Z)**: Alphabetical order by product name
- **Price: Low to High**: Most affordable items first
- **Price: High to Low**: Premium items first
- **Rating**: Highest-rated products first

**How to Sort:**
1. Click the sort dropdown menu
2. Select your preferred sorting option

## 5. Troubleshooting for Customers

### Problem: Cannot Login
**Solutions:**
- Verify email address is entered correctly
- Check password for typos
- Ensure Caps Lock is not enabled
- Password must be at least 6 characters
- Clear browser cache and try again
- Try password reset if available

### Problem: Cannot Add Items to Cart
**Solution:**
- You must be logged in to use the cart
- Click **Login** or **Sign Up** to continue
- If logged in, try refreshing the page
- Check if product is in stock

### Problem: Products Not Loading
**Solutions:**
- Check your internet connection
- Refresh the page (F5 or Ctrl+R)
- Clear browser cache and cookies
- Try a different web browser
- Disable browser extensions temporarily
- Check if JavaScript is enabled

### Problem: Search Returns No Results
**Solutions:**
- Check spelling of search terms
- Use more general keywords (e.g., "phone" instead of specific model)
- Try browsing by category instead
- Use filters to narrow down results
- Remove any active filters that may be limiting results

### Problem: Cart Items Disappear
**Solutions:**
- Ensure you remain logged in
- Check if session has expired (login again)
- Verify products are still in stock
- Clear browser cache and re-add items
- Contact support if issue persists

### Problem: Cannot Complete Checkout
**Solutions:**
- Verify all required fields are filled
- Check that email format is valid
- Ensure delivery address is complete
- Confirm products are still available
- Try different browser if issue persists
- Check internet connection stability

## 3. Product Management

### 3.1 Viewing All Products
1. Click on **Products** tab in admin dashboard
2. Product table displays with columns:
   - Product image thumbnail
   - Product name
   - Category
   - Brand
   - Price (‚Ç±)
   - Stock quantity
   - Status (Active/Inactive)
   - Action buttons (Edit/Delete)
3. Use search box to find specific products
4. Sort by clicking column headers

### 3.2 Adding New Products

#### Step 1: Access Add Product Form
1. Go to **Products** tab
2. Click **Add New Product** button (usually top right)
3. Product form modal or page opens

#### Step 2: Fill Product Information
Complete all required fields marked with asterisk (*):

**Basic Information:**
- **Product Name***: Full name of the product (e.g., "iPhone 15 Pro Max 256GB")
- **Description***: Detailed product description including:
  - Key features
  - What's included
  - Use cases
  - Benefits
- **Category***: Select from dropdown:
  - Smartphones
  - Laptops
  - Monitors
  - Headphones
  - Accessories
- **Brand**: Manufacturer name (e.g., Apple, Samsung, Dell)

**Pricing and Inventory:**
- **Price***: Product price in Philippine Peso (‚Ç±)
  - Enter numbers only
  - System automatically formats currency
- **Stock Quantity**: Number of units available
  - Enter 0 if out of stock
  - System tracks inventory automatically

**Media:**
- **Image URL**: Direct link to product image
  - Use high-quality product photos
  - Recommended size: 800x800 pixels or larger
  - Supported formats: JPG, PNG, WebP
  - Ensure URL is publicly accessible

**Additional Details:**
- **Specifications**: Technical specifications (optional)
  - Can be formatted text
  - Include dimensions, weight, features
- **Rating**: Product rating (0-5 stars)
- **Reviews Count**: Number of customer reviews

#### Step 3: Save Product
1. Review all entered information
2. Click **Add Product** or **Save** button
3. System validates all required fields
4. Product is added to the store
5. Confirmation message appears
6. Product appears in product table

### 3.3 Editing Existing Products

#### Step 1: Locate Product
1. Go to **Products** tab
2. Find the product using:
   - Scroll through the table
   - Use search function
   - Sort by category or name

#### Step 2: Open Edit Form
1. Click **Edit** button for the product
2. Edit form opens with current product data pre-filled
3. All fields are editable

#### Step 3: Make Changes
1. Modify any product information:
   - Update name or description
   - Change price
   - Adjust stock quantity
   - Update category or brand
   - Replace product image
   - Edit specifications
2. All changes are tracked in real-time

#### Step 4: Save Changes
1. Review your modifications
2. Click **Save Changes** button
3. System validates updated information
4. Product information updates in database
5. Confirmation message displays
6. Changes reflect immediately on store front

### 3.4 Deleting Products

**Warning**: Deleting products is permanent and cannot be undone.

#### Deletion Process:
1. Locate the product in the products table
2. Click **Delete** button (usually trash icon)
3. Confirmation dialog appears
4. Review product details to ensure correct selection
5. Click **Confirm Delete** or **Yes, Delete**
6. Product is permanently removed from database
7. Product no longer appears on store front
8. Confirmation message displays

**Important Notes:**
- Deleted products cannot be recovered
- Consider marking products as "Out of Stock" instead
- Check if product has pending orders before deletion
- Product data in completed orders remains intact

### 3.5 Managing Product Inventory

#### Tracking Stock Levels:
1. View stock quantity in products table
2. Products with low stock display warning indicators
3. Out-of-stock products (0 quantity) show special status

#### Updating Inventory:
1. Click **Edit** on product
2. Update **Stock Quantity** field
3. Enter new quantity number
4. Save changes
5. System updates availability status automatically

#### Best Practices:
- Regularly update stock levels
- Monitor low-stock alerts
- Restock popular items promptly
- Set stock to 0 instead of deleting temporarily unavailable items

## 4. Order Management

### 4.1 Viewing All Orders
1. Click **Orders** tab in admin dashboard
2. Orders table displays all customer orders:
   - Order ID/Number
   - Customer name
   - Order date
   - Items ordered
   - Total amount (‚Ç±)
   - Current status
   - Action buttons
3. Orders are sorted by date (newest first)

### 4.2 Order Information Details
Click on any order row to view complete details:
- **Customer Information**:
  - Full name
  - Email address
  - Contact details
- **Shipping Address**:
  - Complete delivery address
  - Postal code
  - Special instructions (if any)
- **Order Items**:
  - Product names and images
  - Quantities ordered
  - Individual prices
  - Subtotals
- **Order Summary**:
  - Subtotal
  - Shipping (if applicable)
  - Total amount
- **Order Status History**:
  - Status changes with timestamps
  - Admin who made changes

### 4.3 Updating Order Status

#### Available Order Statuses:
1. **Pending**: Order just placed, awaiting admin action
2. **Processing**: Order is being prepared/packed
3. **Shipped**: Order dispatched for delivery
4. **Delivered**: Order successfully received by customer
5. **Cancelled**: Order cancelled by customer or admin

#### How to Update Status:
1. Locate the order in orders table
2. Find the **Status** column or dropdown
3. Click on current status dropdown
4. Select new status from list
5. Status updates immediately
6. Customer may receive notification (if configured)
7. Status change is logged with timestamp

#### Status Update Workflow (Recommended):
```
Pending ‚Üí Processing ‚Üí Shipped ‚Üí Delivered
```

**When to Use Each Status:**
- **Pending**: Default for new orders, awaiting your review
- **Processing**: When you start preparing the order
- **Shipped**: When order is handed to courier/delivery service
- **Delivered**: After confirming successful delivery
- **Cancelled**: If order cannot be fulfilled or customer requests cancellation

### 4.4 Processing Orders - Step by Step

#### Step 1: Review New Orders
1. Check **Pending Orders** count on dashboard
2. Go to Orders tab
3. Filter by "Pending" status
4. Review order details thoroughly

#### Step 2: Verify Order Details
For each order check:
- All items are in stock
- Shipping address is complete and clear
- Customer contact information is valid
- Payment status is confirmed (if applicable)

#### Step 3: Begin Processing
1. Gather all items from inventory
2. Update order status to **Processing**
3. Prepare items for shipment
4. Reduce stock quantities if not automatic

#### Step 4: Ship Order
1. Package items securely
2. Generate shipping label
3. Hand over to courier service
4. Obtain tracking number (if available)
5. Update order status to **Shipped**
6. Add tracking information to order notes

#### Step 5: Confirm Delivery
1. Monitor delivery status
2. Once confirmed delivered by courier
3. Update order status to **Delivered**
4. Order is now complete

### 4.5 Handling Order Cancellations

#### Cancellation Reasons:
- Customer request
- Item out of stock
- Unable to contact customer
- Invalid shipping address
- Payment issues

#### Cancellation Process:
1. Locate the order
2. Review cancellation reason
3. Update order status to **Cancelled**
4. If items were reserved, return to inventory
5. Process refund if payment was received
6. Document cancellation reason in notes
7. Contact customer if necessary

### 4.6 Order Search and Filtering

#### Search Options:
- Search by customer name
- Search by order ID/number
- Search by email address
- Filter by date range
- Filter by order status
- Filter by total amount

#### How to Search:
1. Use search box at top of orders table
2. Enter search term
3. Press Enter or click search icon
4. Matching orders display instantly
5. Clear search to view all orders again
